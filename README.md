# 🏆 Hi Sekai TW - 世界計畫台服排名綜合分析平台

歡迎使用 **Hi Sekai TW**！這是一個專為手遊《世界計畫 繽紛舞台！ feat. 初音未來》台服玩家打造的全方位數據分析工具。本平台整合了即時榜單查詢、歷代活動存檔、深度競爭分析以及衝榜資源規劃，旨在為每一位「居民」提供最專業的數據支持。

數據來源：[Hi Sekai API](https://hisekai.org) (非官方粉絲製作)

## 🚀 核心功能介紹

### 1. 🔴 現時活動 (Live Event)
**掌握戰場的最前線。**
*   **即時榜單**：同步查詢 Top 100 玩家的當前分數、時速（1H/3H/24H）及平均得分。
*   **動態安全線 (Safe Line)**：系統會根據活動剩餘時間，自動推算出 Top 100 的「理論安全分數」（超越此分即具備極高機率守住排名）。
*   **死心線 (Give-up Line)**：計算追擊者在剩餘時間內極限運轉也無法超越的分數線，協助調整策略。
*   **精彩片段 (Highlights)**：一鍵切換至特定名次（如 T500, T1000）的看板，觀察大部隊的推進速度。

### 2. 📚 歷代活動 (Archive)
**完整的台服歷史資料庫。**
*   **多維篩選**：支援年份、團體、活動類型（馬拉松/嘉年華/WL）、主角 (Banner)、劇情類型及卡池性質。
*   **快速回顧**：點擊活動卡片即可進入該期結算榜單，觀察該期前百名玩家的最終戰果與分數分佈。

### 3. 🌐 World Link 專屬分析
**針對 WL 特殊機制優化的分析模組。**
*   **輪次過濾**：可切換查看第一輪或第二輪 World Link 的綜合數據。
*   **日均分校正 (chDavg)**：**[重要更新]** 由於 WL 各章節天數不同（如 140 期為 2 天，其餘多為 3 天），系統會自動讀取設定檔中的 `chDavg` 值進行權重校正，提供公平的「日均分」比較。
*   **全域基準切換**：在圖表模式下，可切換 **T100 Base** 或 **T500 Base**，直觀比較各角色章節的競爭熱度。

### 4. 📈 趨勢與比較分析
**用數據解構玩家生態。**
*   **活動比較**：選擇任意兩期活動進行疊加繪圖，系統會自動分析兩者的「競爭陡峭度」與「資源投入度」。
*   **榜線趨勢**：觀察特定名次（如 T1000）隨期數演進的分數變化，掌握台服環境的變遷。
*   **分布概況**：以甘特圖形式呈現角色與團體的活動密度，協助預測未來的 Banner 走向。

### 5. 🕵️ 深度玩家分析
**量化排名的流動性與固化程度。**
*   **活躍玩家排行**：統計歷代活動中上榜次數最多的「常客」，並追蹤其擅長的團體。
*   **排名結構 U(K)**：分析前 K 名的不重複率，判別該區間是「百家爭鳴」還是「名次固化」。

### 6. 🧮 衝榜工具箱
*   **預估資源計算機**：根據歷史相似活動的分數線，輸入您目前的單場 PT 與體力計畫，精確估算所需的「大補充罐」數量（已考慮自然回體）。
*   **玩家狀態查詢**：輸入 ID 即可調閱該玩家的綜合力組成（區域道具、角色等級、家具等）及各難度歌曲通關現況。

## 💡 操作指南

### 如何進行有效的「衝榜預估」？
1.  進入 **「預估資源計算機」**。
2.  **步驟一**：在「參考過往活動」中，篩選出與目標活動團體相同、性質相似的歷史期數（例如：想衝一歌箱活，可參考前幾次 L/n 的箱活數據）。
3.  **步驟二**：選擇「衝榜目標活動」，系統會自動比對兩者的天數差異並修正目標分數。
4.  **步驟三**：填寫您目前打一場的平均得分（請填寫您實際使用的體力倍率），系統將自動推算出總場數與所需飲水。

### 圖表判讀小技巧
*   **斜率越陡**：代表名次間的分數差距極大，存在明顯的斷層，通常發生在極高強度競爭的頭部。
*   **曲線越平緩**：代表該名次區間分數接近，競爭非常激烈，稍有鬆懈名次便會大幅下滑。
